# ウェブページ構成メモ

## 用途
### 今のPC用途
- デュアルブート(80GB割り当て)
- アプリ製作（Unityも含めて）
- Physical AI
- ローカルLLM（8B程度）
- 講義の受講（Zoom）
- ゲーム（ドラクエと音ゲー（adofaiやOsu!））
- 資料作成（MS OfiiceやLaTeX等々）
- 競技プログラミング（AtCoder，PCK）

### 今後行いたいこと
- DTM
- 設計製図（電気・電子回路，CADなど）
- マインクラフト

## 今のスペック
- PC:Dell Insprion 13 1330
- SSD:512GB
- メモリ:16GB(オンボード)
- CPU:Intel Core Ultra5 125H
- GPU:Intel Arc
- 搭載ポート:HDMI×1, USB-A×1，USB-C×2
- その他:日本語配列，キーボードバックライト
- 金額:12万円程度

## 今ある不満
- デュアルブートしてるからストレージが少なすぎる
- オプションやプラグインを入れてるとストレージが足りない
- ローカルLLM使ってるとメモリが少なすぎる
- 同様にグラボ不足
- メモリがオンボードになっているので拡張できない
- 画像認識やPhysical AIの学習をさせるときCUDA（RTXにある）が必要になるが搭載されてない

## 要望
### 最低要望
- ストレージSSD1TB
- メモリ32GB
- CUDA搭載GPU
- 拡張性のあるメモリ
- 22万円程度かそれ以下
- 日本語配列

### 追加要望
- ディスプレイ14インチ以上
- Ryzen CPU
- HDMI，USB-Aが付いている
- キーボードのバックライト

### 購入条件に含めない
- テンキーの有無
- ディスプレイの解像度や種類
